# 🐱 猫娘桌面助手

一个可爱的猫娘桌面AI助手，集成Ollama模型，支持个性化定制和模型微调。

## ✨ 功能特色

### 🎨 视觉设计
- **现代Windows 11美学**：毛玻璃效果、圆角设计、柔和色彩
- **可爱猫娘角色**：精美的动漫风格角色形象
- **流畅动画**：使用Anime.js实现丰富的交互动画
- **个性化主题**：支持多种颜色主题和外观定制

### 🤖 智能对话
- **Ollama集成**：支持多种7B模型（Llama2、Mistral、CodeLlama等）
- **实时对话**：流式响应，自然的打字机效果
- **上下文记忆**：记住对话历史，保持连贯性
- **多模型切换**：可根据需求切换不同的AI模型

### ⚙️ 高级功能
- **模型微调**：支持用户数据训练，定制个性化对话风格
- **可视化训练**：实时显示训练进度和损失曲线
- **数据管理**：支持导入导出对话历史和设置
- **性能优化**：硬件加速、内存限制等高级选项

### 🎯 用户体验
- **桌面集成**：浮动窗口设计，支持拖拽定位
- **快捷键支持**：Ctrl+M快速切换聊天窗口
- **自动隐藏**：可设置空闲时自动隐藏
- **声音提示**：可选的声音反馈系统

## 🚀 快速开始

### 环境要求
- Windows 11 操作系统
- Ollama 已安装并运行
- 现代浏览器（Chrome、Edge、Firefox等）

### 安装步骤

1. **启动Ollama服务**
   ```bash
   ollama serve
   ```

2. **下载所需模型**
   ```bash
   ollama pull llama2
   ollama pull mistral
   # 根据需要下载其他模型
   ```

3. **运行猫娘助手**
   - 双击 `index.html` 文件在浏览器中打开
   - 或者使用本地服务器：
     ```bash
     python -m http.server 8000
     ```

4. **首次使用**
   - 点击右下角的猫娘头像打开聊天窗口
   - 在设置中配置Ollama连接（通常保持默认即可）
   - 选择喜欢的模型开始对话

## 📖 使用指南

### 基本操作
- **打开聊天**：点击猫娘头像或按 Ctrl+M
- **发送消息**：在输入框输入后按Enter或点击发送按钮
- **拖拽移动**：按住猫娘头像可以拖拽到桌面任意位置
- **关闭聊天**：点击聊天窗口外部或再次点击猫娘头像

### 设置配置
点击控制菜单中的"设置"按钮可以打开设置面板：

#### 外观设置
- **主题颜色**：选择喜欢的颜色主题
- **透明度**：调节窗口透明度
- **字体大小**：调整聊天文字大小
- **动画效果**：开关各种动画效果

#### 行为设置
- **猫娘性格**：选择不同的对话风格
- **自动隐藏**：设置空闲时自动隐藏
- **声音提示**：开关声音反馈
- **音量控制**：调节声音大小

#### 高级设置
- **API配置**：设置Ollama连接地址
- **默认模型**：选择启动时使用的模型
- **性能优化**：硬件加速和内存限制
- **数据管理**：导入导出设置

### 模型配置
点击控制菜单中的"模型"按钮可以打开模型配置面板：

#### 模型管理
- **查看可用模型**：显示所有已下载的模型
- **切换模型**：点击选择要使用的模型
- **测试模型**：验证模型是否正常工作

#### 模型微调
1. **准备训练数据**：支持.txt、.json、.csv格式
2. **设置训练参数**：学习率、训练轮数、批次大小等
3. **开始训练**：点击开始按钮进行模型微调
4. **监控进度**：实时查看训练损失和准确率
5. **保存模型**：训练完成后保存自定义模型

## 🔧 故障排除

### 常见问题

#### 无法连接到Ollama
- 确保Ollama服务正在运行：`ollama serve`
- 检查防火墙设置是否允许本地连接
- 确认Ollama API地址设置正确

#### 模型无法加载
- 检查模型是否已下载：`ollama list`
- 确保有足够的磁盘空间
- 尝试重新下载模型

#### 界面显示异常
- 清除浏览器缓存后重试
- 确保使用现代浏览器
- 检查JavaScript是否被禁用

#### 训练失败
- 检查训练数据格式是否正确
- 确保有足够的内存资源
- 尝试调整训练参数

### 获取帮助
- 查看训练日志获取详细错误信息
- 检查浏览器控制台输出
- 确保所有依赖项正确安装
